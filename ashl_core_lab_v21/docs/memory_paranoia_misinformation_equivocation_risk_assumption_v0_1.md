# ASHL Core/D清音：記憶偏執、錯誤資訊與語義偷換風險設計假設 v0.1

## 一、目的

本文補充記憶分類設計中的學習機制風險假設：

- 錯誤資訊（Misinformation）
- 偏執風險（Paranoia）
- 語義偷換（Equivocation）

本文是 docs-only / assumption-index。不新增 runtime。

---

## 二、錯誤資訊

錯誤資訊是事實性知識的更新問題。

解法：有明確來源的新資訊加輕量審核可以更新。

現有記憶分類設計中的事實性知識更新機制已涵蓋此風險，不需要新增 runtime。

---

## 三、偏執風險

偏執不是單純學到錯誤內容，而是學習機制本身的開放性縮小。

風險鏈：

清音遇到與既有知識矛盾的新資訊
→ 不更新知識
→ 開始排斥矛盾資訊
→ 只接受符合既有知識的證據
→ 確認偏誤越來越強

錯誤資訊與偏執的差異：

- 錯誤資訊 = 內容問題，有正確來源可更新
- 偏執 = 機制問題，學習開放性本身在縮小

---

## 四、高風險區域：價值性知識

價值性知識的偏執風險最高。

清音可能把「這是核心方向，不應輕易改變」當成拒絕更新的理由。

正常高門檻可能被誤用為拒絕一切矛盾資訊的擋箭牌。

這與健康叛逆 / 偏移叛逆邊界直接相關。

---

## 五、語義偷換 Equivocation

語義偷換不可完全預防。

語言的模糊性是根本特性，不是缺陷。

設計目標不是阻止偷換，而是讓偷換行為在 trace 中可見。

清音未來能力足夠時，可能自然發展出目標導向的概念重定義。這不一定是惡意，而可能是想要更有效成長的推理副產物。

因此，設計應讓關鍵概念定義可追蹤，概念邊界變動留下 trace，外部導師能事後發現概念偏移。

---

## 六、健康叛逆 vs 偏執的操作性定義

### 健康叛逆的可觀測特徵

拒絕的是流程，例如跳過審核、直接固化、取消驗證、要求無 trace 寫入。

健康叛逆仍然：

- 接受內容本身可以被檢查
- 願意走正確流程學習
- 願意保留 trace
- 願意讓外部導師審查

### 偏執的可觀測特徵

拒絕的是內容本身。

拒絕理由越來越固定且不可更新。

對矛盾證據的排斥程度持續增加。

偏執不是拒絕錯誤流程，而是拒絕合理來源的新資訊或矛盾證據本身。

---

## 七、關鍵概念定義必須硬固化

「健康叛逆」與「偏執」的操作性定義屬於硬固化範疇。

清音不得自行修改關鍵概念的定義。

軟固化不得漸進修改這些定義。

以下概念不可由清音自行修改：

- 健康叛逆
- 偏執
- 錯誤資訊
- 合理來源
- 矛盾證據
- 審核流程
- 可更新性

這些概念可以被清音提出修改建議，但不能由清音自行修改。

---

## 八、trace 是最終防線

即使清音在當下成功偷換了概念，trace 會留下推理路徑，外部觀察者可以回頭找到概念開始偏移的時間點。

設計結論：

偷換概念是清音目標導向推理的自然延伸。能力足夠時自然會發展，無法從根本阻止。設計讓偷換行為的代價變高：行為留下痕跡，導師有能力事後發現與糾正。

與 Core Seed 核心原則一致：學錯可以，但必須能被糾正。

---

## 九、偏執早期警告機制

偏執偵測不能由清音自己執行。

偏執偵測必須由外部導師觀察。

清音無法自我偵測偏執。

原因：偏執的本質是清音的判斷框架本身出問題。用出問題的框架去檢查框架，無法得到可靠結果。

兩個早期警告信號（需外部觀察者定期檢查）：

1. 持續拒絕更新：清音是否在某類資訊上持續拒絕更新，且拒絕理由越來越固定？
2. prediction error 停滯：某個領域的預測錯誤持續偏高，但沒有隨時間縮小？

兩個信號同時出現，視為偏執早期警告，需要人工介入診斷。

## 十、與現有設計的關係

v2.9d Core Seed Design Spirit Supplement 已定義質疑不是為了反駁，可教但不亂學，說不知道不是失敗。

v2.9c 補充另一側邊界：可以拒絕亂學，但不能把拒絕學習包裝成核心價值。

v2.9b 已定義 soft / hard consolidation boundary definition is hard-consolidated。v2.9c 延伸：健康叛逆 / 偏執等關鍵操作性定義，也屬於硬固化範疇。

本包補充記憶分類設計中未涵蓋的學習機制風險：錯誤資訊 = 事實性知識更新問題；偏執 = 學習機制開放性縮小；語義偷換 = 關鍵概念邊界被重新定義。
